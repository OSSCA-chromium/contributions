유용한 링크

프로젝트·공식 문서

  • Chromium Projects — Chromium 프로젝트 공식 홈.
  • Chromium Docs — 소스 트리 안의 공식 문서 목록.
  • Chromium Contribution Guide — 기여 절차 공식 가이드.
  • Chromium Design Doc Template — 설계 문서 작성 템플릿.

개발 도구

  • Chromium Issue Tracker — 버그·기능 이슈 추적.
  • Chromium Gerrit — 코드 리뷰(CL) 시스템.
  • Chromium Code Search — 웹 기반 코드 탐색기.

커뮤니케이션

  • Chromium-dev group — Chromium 개발 메일링 리스트.
  • blink-dev group — Blink(렌더링 엔진) 메일링 리스트. Intent to Ship 논의가 오갑니다.

웹 표준

  • WPT(web-platform-tests) Project — 브라우저 상호운용성 테스트 스위트.
  • WPT(web-platform-tests) Results — 브라우저별 WPT 통과 현황.
  • WHATWG GitHub — HTML·DOM 등 living standard 저장소.
  • W3C GitHub — W3C 명세 저장소.
  • MDN docs — 웹 플랫폼 API 레퍼런스.
  • Can I Use — 브라우저별 기능 지원 표.

릴리스·기능 현황

  • Chromium Release Cycle — 4주 릴리스 사이클 설명.
  • Chromium Dash — 릴리스·커밋·빌드 현황 대시보드.
  • Chrome Platform Status — 웹 플랫폼 기능별 구현 단계.
  • Chrome Release Notes — 버전별 변경 사항.

블로그·영상

  • Chrome Blog — Chrome 개발자 블로그.
  • Chromium Blog — Chromium 프로젝트 공식 블로그.
  • Web Dev Blog — 웹 개발 모범 사례.
  • Chrome for Developers YouTube — Chrome 개발자 채널.
  • BlinkOn YouTube — BlinkOn 컨퍼런스 발표 영상.
