수정한 이슈:

## Summary

컨트리뷰션을 연도(2025/2026)로 구분해서 볼 수 있는 전역 연도 선택기를 추가했다.

- **연도 선택기**: 헤더에서 전체/2026/2025 선택, localStorage로 기억
- **연도별 반응**: 홈·목록·통계·기여자 프로필이 선택 연도에 맞춰 필터·재집계
- **정적 호환**: 필터링은 전부 클라이언트에서 수행, 데이터 스키마 변경 없음

## Background

기여 데이터가 여러 해에 걸쳐 쌓이면서 연도가 섞여 보였다. 특정 해의 활동만 골라 보거나 통계를 연도별로 끊어 보기 어려웠다. 연도 기준 탐색을 한 곳에서 제어할 수 있게 한다.

## Changes

- **연도 선택기**: 헤더에 전역 선택기(전체/2026/2025). 옵션은 데이터에서 동적 생성, 선택은 localStorage 저장
- **연도별 반응**: 홈·목록(/patches)·통계(/stats)·기여자 프로필이 선택 연도에 맞춰 필터·재집계
- **연도 배지**: 각 컨트리뷰션 카드/상세에 연도 배지 표시
- **기본 연도**: 기본 2026, 해당 연도 데이터가 없으면 데이터가 있는 최신 연도(현재 2025)로 자동 폴백 — 첫 화면이 비지 않도록
- **시간대**: 날짜에서 연도 도출, 서울(KST) 기준 분류. 스키마 변경 없음
- **리뷰 반영**: localStorage 차단 환경 크래시 방어, KST 연도 분류, 빈 상태 문구 정리, 단위/통합 테스트 추가
