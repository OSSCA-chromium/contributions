## 유용한 링크

### 프로젝트·공식 문서

- [Chromium Projects](https://www.chromium.org/chromium-projects/) — Chromium 프로젝트 공식 홈.
- [Chromium Docs](https://chromium.googlesource.com/chromium/src/+/main/docs/README.md) — 소스 트리 안의 공식 문서 목록.
- [Chromium Contribution Guide](https://chromium.googlesource.com/chromium/src/+/main/docs/contributing.md) — 기여 절차 공식 가이드.
- [Chromium Design Doc Template](https://docs.google.com/document/d/14YBYKgk-uSfjfwpKFlp_omgUq5hwMVazy_M965s_1KA/edit?tab=t.0#heading=h.7nki9mck5t64) — 설계 문서 작성 템플릿.

### 개발 도구

- [Chromium Issue Tracker](https://issues.chromium.org/u/1/issues) — 버그·기능 이슈 추적.
- [Chromium Gerrit](https://chromium-review.googlesource.com/) — 코드 리뷰(CL) 시스템.
- [Chromium Code Search](https://source.chromium.org/chromium) — 웹 기반 코드 탐색기.

### 커뮤니케이션

- [Chromium-dev group](https://groups.google.com/u/2/a/chromium.org/g/chromium-dev) — Chromium 개발 메일링 리스트.
- [blink-dev group](https://groups.google.com/a/chromium.org/g/blink-dev) — Blink(렌더링 엔진) 메일링 리스트. Intent to Ship 논의가 오갑니다.

### 웹 표준

- [WPT(web-platform-tests) Project](https://github.com/web-platform-tests/wpt/) — 브라우저 상호운용성 테스트 스위트.
- [WPT(web-platform-tests) Results](https://wpt.fyi/results/) — 브라우저별 WPT 통과 현황.
- [WHATWG GitHub](https://github.com/whatwg) — HTML·DOM 등 living standard 저장소.
- [W3C GitHub](https://github.com/w3c) — W3C 명세 저장소.
- [MDN docs](https://developer.mozilla.org/docs/Web) — 웹 플랫폼 API 레퍼런스.
- [Can I Use](https://caniuse.com) — 브라우저별 기능 지원 표.

### 릴리스·기능 현황

- [Chromium Release Cycle](https://chromium.googlesource.com/chromium/src/+/HEAD/docs/process/release_cycle.md) — 4주 릴리스 사이클 설명.
- [Chromium Dash](https://chromiumdash.appspot.com) — 릴리스·커밋·빌드 현황 대시보드.
- [Chrome Platform Status](https://chromestatus.com/) — 웹 플랫폼 기능별 구현 단계.
- [Chrome Release Notes](https://developer.chrome.com/release-notes) — 버전별 변경 사항.

### 블로그·영상

- [Chrome Blog](https://developer.chrome.com/blog) — Chrome 개발자 블로그.
- [Chromium Blog](https://blog.chromium.org/) — Chromium 프로젝트 공식 블로그.
- [Web Dev Blog](https://web.dev/blog) — 웹 개발 모범 사례.
- [Chrome for Developers YouTube](https://www.youtube.com/@ChromeDevs) — Chrome 개발자 채널.
- [BlinkOn YouTube](https://www.youtube.com/@blinkontalks) — BlinkOn 컨퍼런스 발표 영상.
