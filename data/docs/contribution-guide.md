Chromium에 기여하기 전에 행동 강령과 기여 절차, 코드 리뷰 가이드를 숙지하세요.
단계별 실습은 [시작하기](/contributions/docs/getting-started/) →
[첫 CL 올리기](/contributions/docs/first-cl/) →
[코드 리뷰 가이드](/contributions/docs/code-review-guide/) 순서로 읽는 것을
추천하며, 이 문서는 공식 원문 모음입니다.

### Contribution Guide

- [Code of Conduct](https://chromium.googlesource.com/chromium/src/+/main/CODE_OF_CONDUCT.md) — 커뮤니티 행동 강령. 모든 기여자에게 적용됩니다.
- [Contributing](https://chromium.googlesource.com/chromium/src/+/main/docs/contributing.md) — 기여 절차 전반을 다루는 공식 문서.
- [Gerrit Guide](https://chromium.googlesource.com/chromium/src/+/main/docs/gerrit_guide.md) — Gerrit 계정 설정과 사용법.
- [Commit Checklist](https://chromium.googlesource.com/chromium/src/+/main/docs/commit_checklist.md) — CL 업로드 전후로 확인할 체크리스트.

### Code Review

- [Code Reviews](https://chromium.googlesource.com/chromium/src/+/main/docs/code_reviews.md) — OWNERS 제도와 리뷰 정책.
- [CL Respect](https://chromium.googlesource.com/chromium/src/+/main/docs/cl_respect.md) — CL 작성자가 지켜야 할 리뷰 예절.
- [CR Respect](https://chromium.googlesource.com/chromium/src/+/main/docs/cr_respect.md) — 리뷰어가 지켜야 할 리뷰 예절.
- [CL Tips](https://chromium.googlesource.com/chromium/src/+/main/docs/cl_tips.md) — CL을 작게 쪼개고 잘 설명하는 요령.
